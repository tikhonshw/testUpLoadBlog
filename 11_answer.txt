1. git init
2. создать файл .gitignore
2.1. в созданном файле прописать правила согласно задания:
*.sass
/bin
/admin
/config
3. отмечаем все файлы в локальное хранилище: git add .
4. добавляем описание, добавляем в локальное хранилище: git commit -m 'Комментарий для файлов'
5. создаем новую ветку: git branch blog
6. переходим на новую ветку: git checkout blog
